# VLA / 具身智能 Benchmark 重整列表

> 整理口径：基于 `vla_bnchmk.md` 中出现的 benchmark / 评测框架 / 评测平台，并补充当前文档已单独拆出的数据资源类条目。  
> **排序**：两表均按「提出时间」**从新到旧**排列。  
> 其中“Benchmark 主表”收录适合用来做模型评测、横向比较或诊断分析的条目；“数据资源”章节收录更偏训练 / 预训练 / 数据生成的资源，不建议单独作为模型 benchmark 使用。  
> **提出时间**：以论文首次公开（arXiv / 会议）或项目页首次发布为准；平台类条目标注主要版本年份。
